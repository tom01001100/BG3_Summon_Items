# Contributed Configs for Other Screen Resolutions

This folder is for community-contributed `bg3_summon_items_config.ini` files for screen resolutions that are not covered by the default config.

If your setup works at a different resolution, feel free to post your config file here so others can use it.

Suggested filename format:

`bg3_summon_items_config_<width>x<height>.ini`

Example:

`bg3_summon_items_config_3440x1440.ini`
